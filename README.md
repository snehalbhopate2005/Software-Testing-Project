# SwagLabs Login Automation

## Project Overview
Automated login functionality testing for SwagLabs demo website using Selenium WebDriver with Java and TestNG framework.

## Tech Stack
- **Language**: Java
- **Automation Tool**: Selenium WebDriver  
- **Testing Framework**: TestNG
- **Build Tool**: Maven
- **IDE**: Eclipse

## Test Cases Automated
1. **Valid Login Test**: Login with correct username and password
2. **Invalid Login Test**: Login with wrong credentials and verify error message
3. **Locked User Test**: Verify locked_out_user cannot login

## How to Run
1. Clone this repository
2. Open project in Eclipse
3. Run `LoginTest.java` as TestNG Test
